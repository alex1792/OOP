--------------------------------------------------------------------------------
The program uses double ciurular linked list to represent a polygon.

About this program, create two polygon and create a copy polygon of 'a' polygon.
Then check if edges of a and b exists collision or not. If there exists collision,
then print out they encounter collision.

Your input would be like:
1 0 0
1 4 0
2 0 4
1 4 4
-1
1 1 1
1 1 2
1 2 2
1 2 1
-1

First number 1 or 2 depends on what kind of insertion you want. 1 is insert at 
previous of the current node. 2 is insert next to the current node. Then rest of
numbers are the x and y coordinates.
